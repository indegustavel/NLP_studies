# Remoção de stop words.

A remoção de stop words é um processo de remover palavras e termos que não tem significado semântico relevante para a oração, fazendo com que o texto fique mais fácil de ser processado.

Por exemplo, no texto "O cachorro correu para o bosque", as stop words são "o", "para" e "o". Se removemos essas stop words, o texto fica "Cachorro correu bosque.". Para o cérebro humano, talvez perca um pouco a lógica e significado, porém para as máquinas, o texto fica mais fácil de ser processado.
